📖 模式说明：分行 vs 注音

分行模式：将拼音与汉字完全分离。上方为整行拼音，下方为对应汉字。适合长篇阅读或朗诵练习，视觉上更加整洁。

注音模式：采用 HTML5 的 Ruby 标签标准。拼音会紧贴在每个汉字的正上方。适合学习生僻字或精细标注，符合字典排版习惯。

🔒 隐私与安全说明

数据处理：本工具为纯前端应用。所有的汉字转拼音逻辑均通过浏览器本地的 JavaScript (pinyin-pro 库) 完成。您的任何输入内容都不会上传到服务器。

部署信息：网站部署于 GitHub Pages。GitHub 仅提供静态文件托管，由于本站没有任何后端 API 或数据库，您的隐私在物理架构上是绝对安全的。

缓存说明：本站仅在您的浏览器 localStorage 中存储以下两项信息，以便下次访问时恢复您的偏好：
1. pinyin_lang_pref (您的界面语言选择)
2. pinyin_mode_pref (您偏好的显示模式)
